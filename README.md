# 🚲 Fahrrad Raum Website

[Go to English Version](#-fahrrad-raum-website---english)  

Moderne, responsive Website für eine Fahrradreparaturwerkstatt in Braunschweig.  
Erstellt mit React, Tailwind CSS und modernen UI-Komponenten.

---

## ✨ Features

- 📱 Responsive Design (Mobile-first)
- 🛠️ Serviceübersicht (Reparaturen, E-Bike, Inspektion)
- ⭐ Kundenbewertungen
- 📍 Kontaktbereich mit Adresse & Öffnungszeiten
- 🎨 Modernes UI mit Animationen (Framer Motion)

---

## 🧰 Tech Stack

- React / Next.js
- Tailwind CSS
- shadcn/ui Komponenten
- Lucide Icons
- Framer Motion

---

## 🚀 Installation

1. [GitHub Repository](https://github.com/Noxx5/fahrrad-raum) klonen:

```bash
git clone https://github.com/Noxx5/fahrrad-raum.git
```

2. Projekt öffnen:

```bash
cd fahrrad-raum
```

3. Packages Installieren:

```bash
pnpm i
```

4. Projekt starten:

```bash
pnpm dev
```

5. Website öffnen:
[http://localhost:3000](http://localhost:3000)

---

## 📂 Projektstruktur

```bash
/src                    # Projekt Ordner
    /app                # App Ordner
        page.tsx        # Hauptseite
    /components         # UI Komponenten
        /ui             # Shadcn Components
/public                 # Bilder & Assets
```

# 🚲 Fahrrad Raum Website - English

A modern, responsive website for a bicycle repair shop in Braunschweig.  
Built with React, Tailwind CSS, and modern UI components.

---

## ✨ Features

- 📱 Responsive design (mobile-first)
- 🛠️ Service overview (repairs, e-bikes, inspections)
- ⭐ Customer reviews
- 📍 Contact section with address & hours of operation
- 🎨 Modern UI with animations (Framer Motion)

---

## 🧰 Tech Stack

- React / Next.js
- Tailwind CSS
- shadcn/ui components
- Lucide Icons
- Framer Motion

---

## 🚀 Installation

1. Clone the [GitHub Repository](https://github.com/Noxx5/fahrrad-raum):

```bash
git clone https://github.com/Noxx5/fahrrad-raum.git
```

2. Open the project:

```bash
cd fahrrad-raum
```

3. Install packages:

```bash
pnpm i
```

4. Start the project:

```bash
pnpm dev
```

5. Open the website:
[http://localhost:3000](http://localhost:3000)
